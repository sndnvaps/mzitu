# mzitu
webspider for crawl the mm pic from mzitu.com

# how to get it 

```bash
 go get github.com/sndnvaps/mzitu
```

# download the pic to local path

In Linux system
```bash
  cd $GOPATH/src/github.com/sndnvaps/mzitu
  go build 
  ./mzitu
```

In Windows system
```bash
  cd %GOPATH%\src\github.com\sndnvaps\mzitu
  go build
  .\mzitu
```
